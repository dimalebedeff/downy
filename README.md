# Downy

Расширение Chrome для скачивания видео с любых сайтов: находит видео и аудио на страницах, а нативное companion-приложение (CoApp) скачивает и склеивает стримы через ffmpeg, для сложных сайтов используя yt-dlp.

## Что умеет

- Детект медиа на вкладке: прямые файлы (mp4, webm, mp3 и др.) и HLS-стримы (m3u8) — по сетевым запросам и по DOM.
- Бейдж-счётчик на иконке, попап со списком: превью-кадр, название, качество (варианты из мастер-плейлиста), размер/длительность. Если превью нет на странице, CoApp вытаскивает кадр через ffmpeg.
- Все загрузки идут через CoApp: прямые файлы качает сам CoApp, HLS склеивается в mp4 через ffmpeg. Единый список загрузок с шкалой, скачанными МБ/ГБ (для HLS полный размер оценивается) и отменой.
- Кнопка «Скачать страницу через yt-dlp» — запасной экстрактор для сложных сайтов.
- DRM-контент (Netflix и т.п.) не поддерживается принципиально.

## Установка (Windows)

1. **Установи Node.js**, если его ещё нет: [nodejs.org](https://nodejs.org) → зелёная кнопка **LTS** → запусти установщик, везде жми «Далее».
2. **Скачай проект**: на [странице релизов](https://github.com/dimalebedeff/downy/releases/latest) скачай архив **Source code (zip)** из раздела Assets и распакуй его в любую постоянную папку (например, `D:\downy`). Не удаляй её потом — браузер запускает расширение и CoApp прямо из неё.
3. **Запусти `install.bat`** (двойной клик) и дождись надписи «Готово!». Скрипт сам скачает всё нужное и зарегистрирует CoApp. Если Windows предупредит о неизвестном файле — «Подробнее» → «Выполнить в любом случае».
4. **Подключи расширение в Chrome**: набери в адресной строке `chrome://extensions`, включи справа сверху **«Режим разработчика»**, нажми **«Загрузить распакованное расширение»** и выбери папку `extension\dist` внутри папки проекта.
5. **Перезапусти браузер.**

Проверка: нажми на иконку Downy — статус «CoApp» в шапке попапа должен быть зелёным.

## Как пользоваться

1. Открой страницу с видео и запусти воспроизведение — на иконке Downy появится счётчик найденного.
2. Нажми на иконку: в списке будут найденные видео с превью, качеством и размером. Жми «Скачать».
3. Если список пуст (сайт хитрит) — кнопка «Скачать страницу через yt-dlp» внизу попапа.
4. Файлы сохраняются в `Downloads\downy`. Поменять папку: «Настройки» внизу попапа → поле пути или кнопка «Обзор…».

## Архитектура

```
extension/   MV3-расширение (TypeScript, esbuild)
  src/background.ts   сниффинг webRequest, m3u8-парсинг, связь с CoApp
  src/content.ts      детект <video>/<audio> в DOM
  src/popup/          UI списка и загрузок
coapp/       нативный хост (Node.js, Native Messaging)
  src/host.ts         задания: HLS через ffmpeg, прямые файлы, yt-dlp, кадры-превью
  install.mjs         регистрация native messaging host в реестре
  fetch-bins.mjs      загрузка ffmpeg/yt-dlp в coapp/bin
shared/protocol.ts    типы сообщений расширение ↔ CoApp
```

## Ограничения (пока)

- Только Windows и Chromium-браузеры (Chrome, Edge). Firefox — позже.
- DASH (mpd) не поддерживается — следующий этап.
- Заголовки для ffmpeg передаются только Referer и User-Agent; куки не передаются — редкие сайты с куки-защитой сегментов не скачаются (обычно спасает yt-dlp).
