# YaDaun

Расширение Chrome для скачивания видео с любых сайтов: находит видео и аудио на страницах, а нативное companion-приложение (CoApp) скачивает и склеивает стримы через ffmpeg, для сложных сайтов используя yt-dlp.

## Что умеет

- Детект медиа на вкладке: прямые файлы (mp4, webm, mp3 и др.), HLS-стримы (m3u8) и DASH (mpd) — по сетевым запросам и по DOM.
- Попап со списком: превью-кадр, название, качество (варианты из мастер-плейлиста), размер/длительность. Если превью нет на странице, CoApp вытаскивает кадр через ffmpeg.
- Все загрузки идут через CoApp: HLS качается параллельным yt-dlp (быстро даже на троттлящих CDN вроде ВК), прямые файлы — сам CoApp. Единый список загрузок с шкалой, скачанными МБ/ГБ и отменой.
- Кнопка «Скачать страницу через yt-dlp» — запасной экстрактор для сложных сайтов (YouTube и др.).
- Выбор дорожек: видео+звук, только видео или только звук — и для найденного медиа, и для yt-dlp.
- Самообновление: попап проверяет релизы на GitHub (раз в 6 часов) и предлагает кнопку «Обновить» — скачивание, пересборка и перезапуск расширения в один клик; заодно обновляется yt-dlp.
- DRM-контент (Netflix и т.п.) не поддерживается принципиально.

## Установка (Windows)

1. **Установи Node.js**, если его ещё нет. Проще всего одной командой в PowerShell (Пуск → набери «PowerShell» → Enter):
   ```powershell
   winget install OpenJS.NodeJS.LTS
   ```
   Либо вручную: [nodejs.org](https://nodejs.org) → зелёная кнопка **LTS** → запусти установщик, везде жми «Далее».
2. **Скачай проект**: на [странице релизов](https://github.com/dimalebedeff/downy/releases/latest) скачай архив **Source code (zip)** из раздела Assets и распакуй его в любое постоянное место, например на диск `D:`. Внутри окажется папка вида `downy-0.1` — это папка проекта, не удаляй её после установки: расширение и CoApp работают прямо из неё.
3. **Запусти `install.bat`** из папки проекта (двойной клик) и дождись надписи «Готово!». Скрипт сам скачает всё нужное, соберёт расширение и зарегистрирует CoApp. Если Windows предупредит о неизвестном файле — «Подробнее» → «Выполнить в любом случае».
4. **Подключи расширение в Chrome**: набери в адресной строке `chrome://extensions`, включи справа сверху **«Режим разработчика»**, нажми **«Загрузить распакованное расширение»** и выбери папку `extension\dist` внутри папки проекта (она появляется после шага 3).
5. **Перезапусти браузер.**

Проверка: нажми на иконку YaDaun на странице с видео — точка-индикатор в шапке попапа загорится зелёным (по клику на неё — состояние помощника).

## Как пользоваться

1. Открой страницу с видео и запусти воспроизведение.
2. Нажми на иконку YaDaun: найденные видео покажутся карточками с превью, качеством и размером. Жми «Скачать» — прогресс идёт прямо в карточке; «только видео» и «только звук» — в меню «⋮».
3. Если по сети ничего не поймалось, но на странице крутится видео (YouTube и т.п.) — покажется карточка страницы, качающая через yt-dlp. На совсем пустой вкладке — кнопка «Скачать страницу через yt-dlp».
4. Файлы сохраняются в `Downloads\downy`. Поменять папку: шестерёнка в шапке попапа → поле пути или кнопка «Обзор…».

## Архитектура

```
extension/   MV3-расширение (TypeScript, esbuild)
  src/background.ts   сниффинг webRequest, m3u8-парсинг, связь с CoApp
  src/content.ts      детект <video>/<audio> в DOM
  src/popup/          UI списка и загрузок
coapp/       нативный хост (Node.js, Native Messaging)
  src/host.ts         задания: HLS через ffmpeg, прямые файлы, yt-dlp, кадры-превью
  install.mjs         регистрация native messaging host в реестре
  fetch-bins.mjs      загрузка ffmpeg/yt-dlp в coapp/bin
shared/protocol.ts    типы сообщений расширение ↔ CoApp
```

## Ограничения (пока)

- Только Windows и Chromium-браузеры (Chrome, Edge). Firefox — позже.
- Заголовки для ffmpeg передаются только Referer и User-Agent; куки не передаются — редкие сайты с куки-защитой сегментов не скачаются (обычно спасает yt-dlp).
